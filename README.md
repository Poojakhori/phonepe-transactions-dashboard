# PhonePe Transactions Dashboard (2024)

## Tools Used
- Power BI
- Excel

## Project Description
This dashboard analyzes PhonePe transaction data for 2024 to identify
transaction trends, payment distribution, and key performance indicators.

## Key Insights
- Highest transactions in Q3
- UPI dominated total payments
- Karnataka had highest transaction volume
